SQL Soruları ve Cevapları

Soru 1:

Write a query that returns track name and its composer from tracks table.

Cevap 1:

SELECT Name, Composer FROM track

Bu sorgu, şarkı adını ve bestecisini "tracks" tablosundan getirir.

Soru 2:

Write a query that returns the unique name of composers of each track.

Cevap 2:

SELECT DISTINCT Composer FROM track

Bu sorgu, her şarkının bestecisinin benzersiz adını getirir.

Soru 3:

Write a query that returns all columns from the tracks table.

Cevap 3:

SELECT * FROM track

Bu sorgu, "tracks" tablosundaki tüm sütunları getirir.

Soru 4:

Write a query that returns the unique combinations of AlbumId and MediaTypeId from the tracks table.

Cevap 4:

SELECT DISTINCT AlbumId, MediaTypeId FROM track

Bu sorgu, "tracks" tablosundaki AlbumId ve MediaTypeId sütunlarının benzersiz kombinasyonlarını getirir.

Soru 5:

Write a query that returns track name and track ID for tracks composed by 'Jorge Ben'.

Cevap 5:

SELECT Name, TrackID FROM Track WHERE Composer = 'Jorge Be';

Bu sorgu, 'Jorge Ben' tarafından bestelenen şarkıların adını ve ID'sini getirir.

Soru 6:

Write a query that returns all information of the invoices of which total amount is greater than $25.

Cevap 6:

SELECT * FROM invoice WHERE Total > 25

Bu sorgu, toplam tutarı $25'ten büyük olan faturaların tüm bilgilerini getirir.

Soru 7:

Write a query that returns all information of the first 5 invoices where the total amount is less than $15.

Cevap 7:

SELECT * FROM invoice WHERE Total < 15 LIMIT 5;

Bu sorgu, toplam tutarı $15'ten düşük olan ilk 5 faturanın tüm bilgilerini getirir.

Soru 8:

Write a query that returns all information of the invoices of which total amount is greater than $10, sorted from highest to lowest total amount, and displaying the top 2 rows.

Cevap 8:

SELECT * FROM invoice WHERE Total > 10 ORDER BY Total DESC LIMIT 2;

Bu sorgu, toplam tutarı $10'dan büyük olan faturaların toplam tutarına göre yüksekten düşüğe sıralanmış ilk 2 satırını getirir.

Soru 9:

Write a query that returns all information of the invoices where the billing country is not Canada, sorted from highest to lowest total amount, and displaying the top 10 rows.

Cevap 9:

SELECT * FROM invoice WHERE BillingCountry <> 'Canada' ORDER BY Total DESC LIMIT 10;

Bu sorgu, fatura ülkesi Kanada olmayan faturaların toplam tutarına göre yüksekten düşüğe sıralanmış ilk 10 satırını getirir.

Soru 10:

Write a query that returns the artist's album information.

Cevap 10:

Bu sorgu, sanatçının albüm bilgilerini getirmek için birden fazla tabloyu içeren bir SQL sorgusu kullanır. Varsayılan olarak "artists," "albums," ve olası bir ara tabloyu (örneğin, "artist_albums") sanatçılar ve albümler arasındaki ilişkiyi temsil etmek için kullanıyorsanız, işte genel bir sorgu yapısı:

SELECT a.ArtistName, al.AlbumName, al.ReleaseYear FROM artists a JOIN artist_albums aa ON a.ArtistId = aa.ArtistId JOIN albums al ON aa.AlbumId = al.AlbumId;

Bu sorgu, sanatçının albüm bilgilerini getirir.

Soru 11:

Write a query that returns the track names that start with 'B' and end with 'S'.

Cevap 11:

Bu sorgu, 'B' ile başlayan ve 'S' ile biten şarkı adlarını getirir.

Soru 12:

Write a query that returns the newest date among the invoice dates between 2008 and 2011.

Cevap 12:

Bu sorgu, 2008 ile 2011 tarihleri arasındaki fatura tarihleri arasındaki en yeni tarihi getirir.

Cevap 17:

Soru 19:

Write a query that returns the minimum duration of a track in each album, displaying album ID, album title, and duration of the track.

Cevap 19:

SELECT a.AlbumId, a.Title AS AlbumTitle, MIN(t.Milliseconds) AS MinimumDuration FROM albums a JOIN tracks t ON a.AlbumId = t.AlbumId GROUP BY a.AlbumId, a.Title;

Bu sorgu, her albümdeki bir şarkının minimum süresini, albüm ID'sini, albüm başlığını ve şarkının süresini getirir.

Soru 20:

Write a query that returns albums whose total duration is higher than 60 minutes, displaying album title and their durations, sorted from highest to lowest duration.

Cevap 20:

SELECT a.Title AS AlbumTitle, SUM(t.Milliseconds) / 60000 AS DurationInMinutes FROM albums a JOIN tracks t ON a.AlbumId = t.AlbumId GROUP BY a.AlbumId, a.Title HAVING DurationInMinutes > 60 ORDER BY DurationInMinutes DESC;

Bu sorgu, toplam süresi 60 dakikadan fazla olan albümleri, albüm başlığı ve süreleri ile birlikte getirir ve süreye göre en yüksekten en düşüğe sıralar.

Soru 21:

Write a query that returns the track ID, track name, and album ID info of the albums with titles 'Prenda Minha', 'Heart of the Night', and 'Out Of Exile'.

Cevap 21:

SELECT t.TrackId, t.Name AS TrackName, t.AlbumId FROM tracks t JOIN albums a ON t.AlbumId = a.AlbumId WHERE a.Title IN ('Prenda Minha', 'Heart of the Night', 'Out Of Exile');

Bu sorgu, 'Prenda Minha', 'Heart of the Night' ve 'Out Of Exile' başlıklı albümlerin parça ID'sini, parça adını ve albüm ID bilgisini getirir.

Soru 22:

Write a query that returns the track names having the minimum and maximum duration, separately.

Cevap 22:

Minimum Duration:
SELECT Name AS TrackName, Milliseconds AS Duration FROM tracks ORDER BY Duration ASC LIMIT 1;
Bu sorgu, en kısa süreli şarkının adını ve süresini getirir.

Maximum Duration:
SELECT Name AS TrackName, Milliseconds AS Duration FROM tracks ORDER BY Duration DESC LIMIT 1;
Bu sorgu, en uzun süreli şarkının adını ve süresini getirir.

Soru 23:

Write a query that returns the tracks having a duration less than the average duration.

Cevap 23:

SELECT * FROM tracks WHERE Milliseconds < ( SELECT AVG(Milliseconds) FROM tracks );
Bu sorgu, ortalama süreden daha kısa süreli olan tüm şarkıları getirir.
