Node.js Nedir? - What is Node.js?

Node.js, açık kaynaklı bir, sunucu tarafında JavaScript çalıştırma ortamıdır ve geliştiricilere sunucu tarafında JavaScript kodlarını çalıştırma olanağı sağlar.

Node.js is an open-source, server-side JavaScript runtime environment that allows developers to run JavaScript code on the server.

İşte Node.js hakkında ve bununla neler yapabileceğiniz konusunda önemli noktalar:

Here are some key aspects of Node.js and what you can do with it:

  • Sunucu Tarafı JavaScript: Node.js, sunucu tarafı betik yazmak için JavaScript kullanmanıza olanak tanır. Bu, geliştiricilerin hem istemci tarafında hem de sunucu tarafında aynı dilde kod yazabilmesi anlamına gelir, bu da tam yığın geliştiriciler için geliştirmeyi basitleştirir.
  • Server-Side JavaScript: Node.js allows you to use JavaScript for server-side scripting, which was traditionally the realm of languages like PHP, Ruby, and Python. This means you can write both client-side and server-side code in the same language, simplifying development for full-stack developers.
  • Olay Tabanlı ve Engelleme Olmayan G/Ç: Node.js, engelleme yapmayan ve olay tabanlı olarak tasarlanmıştır, bu da ölçeklenebilir ve gerçek zamanlı uygulamaları oluşturmak için oldukça uygundur. İplikler gerektirmeksizin büyük sayıda eşzamanlı bağlantıyı yönetebilir, bu da verimli ve yüksek performanslı hale getirir.
  • Event-Driven and Non-Blocking I/O: Node.js is designed to be non-blocking and event-driven, which makes it particularly suitable for building scalable and real-time applications. It can handle a large number of concurrent connections without the need for threads, making it efficient and highly performant.
  • Paket Ekosistemi: Node.js, Node Package Manager (NPM) aracılığıyla kullanılabilir birçok paket ve modülün bulunduğu zengin bir ekosisteme sahiptir. Bu geniş kütüphane deposu, web geliştirmeden veri işlemeye kadar çeşitli görevler için önceden oluşturulmuş modüller sağlayarak geliştirme sürecini basitleştirir.
  • Package Ecosystem: Node.js has a rich ecosystem of packages and modules available through the Node Package Manager (NPM). This vast repository of libraries simplifies the development process by providing pre-built modules for various tasks, from web development to data processing.
  • Çoklu Platform: Node.js, Windows, macOS ve Linux gibi çeşitli işletim sistemlerinde çalışabilir ve farklı sunucu ortamları için çok yönlüdür.
  • Cross-Platform: Its cross-platform compatibility allows developers to use it on various operating systems.
  • Web Sunucular: Node.js, genellikle web sunucuları oluşturmak için kullanılır. Popüler web çerçeveleri, özellikle Express.js, güçlü ve ölçeklenebilir web uygulamaları oluşturmayı kolaylaştırır.
  • Web Servers: Node.js is often used to build web servers. Popular web frameworks like Express.js make it easy to create robust and scalable web applications.
  • API Geliştirme: Node.js, verimliliği ve yüksek API isteklerini işleme yeteneği nedeniyle API (Uygulama Programlama Arabirimi) oluşturmak için oldukça uygundur.
  • API Development: Node.js is well-suited for building APIs (Application Programming Interfaces) because of its efficiency and the ability to handle a high volume of API requests.
  • Gerçek Zamanlı Uygulamalar: Node.js, olay tabanlı mimarisi sayesinde sohbet uygulamaları, çevrimiçi oyunlar, işbirlikçi araçlar ve akış hizmetleri gibi gerçek zamanlı uygulamaları oluşturmak için yaygın olarak kullanılır.
  • Real-Time Applications: It excels at building real-time applications and services, making it ideal for chat, gaming, and streaming platforms.
  • Mikro Hizmetler: Birçok organizasyon, büyük uygulamaları oluşturmak için bir araya gelerek bağımsız ve ölçeklenebilir hizmetler inşa etmek için Node.js'i kullanır.
  • Microservices: Many organizations adopt Node.js for building microservices, which are small, independent, and scalable services that work together to create larger applications.
  • Komut Satırı Araçları: Node.js, çeşitli otomasyon görevleri için komut satırı araçları ve betikler oluşturmak için kullanılabilir.
  • Command-Line Tools: Node.js can be used to build command-line tools and scripts for various automation tasks.
  • Topluluk Desteği: Node.js, büyük ve aktif bir geliştirici topluluğuna sahiptir, bu da bol miktarda kaynak, belge ve destek sağlar.
  • Community Support: Node.js has a large and active community of developers, which means plenty of resources, documentation, and support are available.
  • Çoklu Platform: Çeşitli işletim sistemlerinde kullanılabilmesi, Node.js'i farklı işletim sistemlerinde kullanmak isteyenler için uygundur.
  • Cross-Platform: Its cross-platform compatibility allows developers to use it on various operating systems.

  8. Scalability: It is designed for building scalable applications that can handle a high volume of concurrent connections.

  10. Real-Time Capabilities: It excels at building real-time applications and services, making it ideal for chat, gaming, and streaming platforms.

Person and BankAccount Example

class Person {
constructor(name) {
this.name = name;
}
sayName() {
console.log(`My name is ${this.name}`);
}
}

class BankAccount extends Person {
#balance;

constructor(name, balance) {
super(name);
this.#balance = balance;
}

deposit(amount) {
if (amount > 0) {
this.#balance += amount;
console.log(`${this.name} deposited $${amount}`);
} else {
console.log("Invalid deposit amount");
}
}

withdraw(amount) {
if (amount > 0 && amount <= this.#balance) {
this.#balance -= amount;
console.log(`${this.name} withdrew $${amount}`);
} else {
console.log("Invalid withdrawal amount or insufficient balance");
}
}

getBalance() {
console.log(`${this.name}'s balance is $${this.#balance}`);
}
}

// Example usage:
const person = new Person("John");
person.sayName();

const account = new BankAccount("Alice", 1000);
account.sayName();
account.deposit(500);
account.getBalance();
account.withdraw(200);
account.getBalance();
